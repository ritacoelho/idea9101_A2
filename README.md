# idea9101_A2

### To run from project root:

```console
npm install
npm start
```

open browser in `localhost:3300`

open browser in `localhost:3000`

---

### To run it manually or individually, you can open two terminal windows:

##### Terminal 1:
```console
cd phone_mqtt_sender
npm install
npm start
```

open browser in `localhost:3300`


##### Terminal 2:
```console
cd desktop_mqtt_receiver
npm install
npm start
```

open browser in `localhost:3000`
